COMO INSTALAR O PAINEL SDR COMO APP
=====================================

PASSO 1 — Criar o repositório no GitHub
-----------------------------------------
1. Acesse: github.com
2. Clique no "+" no canto superior direito → "New repository"
3. Nome do repositório: painel-sdr
4. Deixe como "Public"
5. NÃO marque nenhuma opção extra
6. Clique em "Create repository"

PASSO 2 — Subir os arquivos
-----------------------------
Na página do repositório vazio que abriu, clique em:
"uploading an existing file"

Arraste TODOS esses arquivos de uma vez:
- index.html
- manifest.json
- sw.js
- icon-192.png
- icon-512.png

Clique em "Commit changes" (botão verde lá embaixo)

PASSO 3 — Ativar o GitHub Pages
---------------------------------
1. Clique em "Settings" (no menu do repositório)
2. No menu lateral esquerdo, clique em "Pages"
3. Em "Source", selecione "Deploy from a branch"
4. Em "Branch", selecione "main" e pasta "/ (root)"
5. Clique em "Save"
6. Aguarde 1-2 minutos

Seu app estará em:
https://yhenrique-second.github.io/painel-sdr/

PASSO 4 — Instalar como app no Windows
----------------------------------------
1. Abra o link acima no Google Chrome
2. Na barra de endereço, vai aparecer um ícone de
   monitor com uma setinha (canto direito da barra)
3. Clique nele → "Instalar Painel SDR"
4. Pronto! Vai aparecer como app no seu Desktop e
   no menu Iniciar do Windows

A partir daí é só abrir pelo ícone, sem navegador!
